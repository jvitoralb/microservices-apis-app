Usage

Post a URL like: https://www.google.com

Use the code received as {... "short_url": 44451} to be redirected to the URL posted

this.app.com/url-shortener/api/shorturl/44451

Note:

Don't forget to use "/api/shorturl/short_url"

Expected Output

{"original_url": "https://www.google.com", "short_url": 44451}
