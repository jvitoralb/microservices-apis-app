{ "username": "harpyja", "_id": "63012a9c8541ec94d7ab1e54" }

To get all users go to:

this.app.com/api/users

{ "_id": "63012a9c8541ec94d7ab1e54",

"duration": 30, "date": "Sat Aug 20 2022" }

Logs

Get Logs

Usage

You can also get your exercises Logs using the Logs form:

{ "username": "harpyja", "count": 1, "_id": "63012a9c8541ec94d7ab1e54", "log": [ { "description": "Walk", "duration": 30, "date": "Sat Aug 20 2022" } ] }

Set the url with _id /api/user/63012a9c8541ec94d7ab1e54/logs

Notes

You can also set a limit, from and to.

  • Limit: integer to set how many logs the log array should have.
  • From&To: dates in format YYYY-MM-DD to only show the exercises in the log array within the provided date interval.

Use GET /api/users/:_id/logs?[from][&to][&limit]

Don't forget to use this.app.com/exercise-tracker when setting URL's
